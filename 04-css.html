CSS

Cascading Style Sheets

CSS is a language used to describe the presentation of markup.

Applying CSS

  • style attribute (inline styles)
  • <style> element
  • external style sheet

style attribute

style is a global attribute.

Using the attribute will override styles declared in a <style> tag or a external style sheet.

<p style="color: red;">This is a sentence</p>

<style> element

The <style> element is included in the <head> of a document.

<style>
  /* styles go here */
</style>

External Style Sheet

CSS can be stored by itself in a file.

Within the <head> of each HTML file, link to the style sheet file.

<link href="path/to/css/style.css" rel="stylesheet" />

All CSS rules in this file will apply to any HTML document which links to it.

Which method to use?

If a style you plan to use will be repurposed multiple times, consider adding it to a <style> element on the page or an external style sheet.

<style>
  circle { fill: red; }
</style>

If the style will change briefly (through interactivity), it is okay to use the style attribute.

CSS Rules

selector { property: value; }

Selectors

The selector identifies the part of HTML to be styled.

  • type (body, p, abbr)
  • id (#preamble)
  • class (.branches-of-gov)
  • descendant (nav li)
  • child (ul > li)
  • adjacent sibling (h1 + p)
  • general sibling (p ~ p)
  • attribute (abbr[title])

Pseudo-classes

Pseudo-classes refer to the state of an element -- how is it interacted with by a user or the Document Object Model (DOM).

These are preceded by a single colon.

  • :hover: when a user hovers over the element
  • :first-child: the first element in a group of siblings
  • :visited: a link which has been visited

a:hover { color: navy; }

Pseudo-elements

Pseudo-elements select very specific parts of an element. These are preceded by a double colon (::).

  • ::after: space after an element
  • ::before: space before
  • ::first-letter: first character
  • ::first-line: first line

p::first-line { font-weight: bold; }

Declaration

A declaration (all enclosed within the curly braces { }) is comprised of a property and value, and ends by a semi-colon (;).

selector { property: value; }

A selector can have multiple declarations.

selector {
  property: value;
  property2: value2;
}

Properties & Values

A property and its value are separated by a colon (:).

There are hundreds of properties. Values depend on the property.

SVG & CSS

There are a few properties unique to styling SVG:

  • fill: the color of the element
  • stroke: the color of the element's outline
  • stroke-width: the width of the element's outline
